## Dependencies

- Python 3.6 <br/>
- Fastai 1.0.52 <br/>
- Flask <br/>
- Gunicorn

## Instructions
First run `sudo pip install -r requirements.txt` to install the required dependencies. Then launch the app by running `python app.py`. Open up your browser and navigate to http://localhost:8008
